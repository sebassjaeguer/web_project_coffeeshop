# Coffee Shop - Triple Espresso

Este proyecto es una página web diseñada para una cafetería llamada Triple Espresso.  
El objetivo del proyecto es la practica del sprint 4 de triple ten con lo aprendido durante este

# Tecnologías

Se Utilizo:

- HTML5 para la pagina principal
- CSS para dar estilos

# partes del proyecto

# encabezado

- Menú de navegación
- Imagen decorativa
- Título y descripción

# recetas

- Videos integrados desde YouTube mediante iframes
- Subtítulos y diseño organizado

# reservacion con formulario

- Campos para nombre, correo, fecha, hora y número de personas
- Checkbox para aceptar términos de uso
- Botón submit

# footer

- Círculo decorativo
- Logo colocado con medidas exactas
- Sección de redes sociales
